Bulls & Cows

Guide for BUlls & Cows Game :

The system will select a secret number contain 4 different digits.
In each round try to guess the secret number.
For each number you guess correctly but it is not placed correctly B will increase by one.
For each number you guess correctly and it is placed correctly A will increase by one.
When you reach the exact number you will win the game.
The maximum number of points you can earn in a single game is 10, for each game you will get 10 attempts, if you can not guess correctly by the end of the game your score will be 0
